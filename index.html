Preguntas frecuentes

Podés realizar el pago de tu página Web por Mercado Pago o por transferencia bancaria.

Necesitamos un mínimo de 24 hs para crear la página perfecta. Una vez que realicés el pedido de la página web, vamos a contactarnos con vos para coordinar cómo diseñarla. Cuando esté lista te la mostraremos. Sólo queda realizar el pago y ya podés enviarla como regalo.

Lo que vos quieras. Vamos a enviarte una lista con las opciones de diseño y vos vas a seleccionar las fotos, palabras y música que quieras que pongamos en la Web.

El dominio Web es adquirido por 1 año. Vamos a comprar por vos un dominio que tenga tu nombre y el de tu pareja (www.SuNombreYTuNombre.com.ar). Cuando finalice el año podés pedirnos gestionar la renovación del mismo para que éste siga en la Web (debés pagar la renovación si así lo querés). También podés pedirnos que lo demos de baja en cualquier momento o que te cedamos el derecho del dominio (estas acciones no tienen costo).

Solamente podrá verla la persona que conozca la dirección Web. Podes compartirle el link a amigos y familiares para que vean el regalo. Si querés que sólo vos y la persona que recibe el regalo puedan entrar y ver el contenido, podes pedirnos que pongamos una contraseña para que ésta sea privada.
